# 🚀 Day 11 - Scheduled Workflow

## 📌 Overview
This workflow demonstrates how to run jobs automatically at scheduled times using cron syntax in GitHub Actions.

## 📂 Directory Structure
```bash
30-Days-of-GitHub-Actions/
├── .github/
│   └── workflows/
│       └── day-11.yml
└── Day-11/
    └── README.md
```

## ⚙️ What it does
- Runs automatically on a schedule (daily)
- Supports manual trigger
- Prints date and messages
- Simulates scheduled tasks

## 📂 Files
- `.github/workflows/day-11.yml` → Workflow definition  
- `Day-11/README.md` → Documentation  

## ▶️ Key Steps
1. Define cron schedule  
2. Enable manual trigger (`workflow_dispatch`)  
3. Execute scheduled job  
4. Print logs  

## 🎯 Purpose
To understand:
- Cron syntax in GitHub Actions  
- Scheduled automation  
- Manual vs automatic triggers
