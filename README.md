# 🚀 30 Days of GitHub Actions

## 📌 Overview
This repository contains a 30-day hands-on journey of learning GitHub Actions.  
Each day focuses on a specific concept, starting from basics to advanced CI/CD workflows.

The goal is to build consistency, understand real DevOps practices, and showcase practical pipeline implementations.

---

## 🧠 What This Covers
- Workflow basics and triggers  
- Jobs, steps, and conditions  
- Environment variables and secrets  
- Artifacts and caching  
- Matrix builds  
- Reusable workflows  
- Docker integration  
- CI/CD pipelines  
- Deployment strategies  
- Failure handling and rollback
